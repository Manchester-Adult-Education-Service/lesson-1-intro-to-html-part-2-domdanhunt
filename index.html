Project Showcase: Library System

This software was developed to help local libraries manage their book collections and borrower records more effectively using Python logic.

Core Features

  • Record new book loans
  • View all active records
  • Search by borrower name
  • Automated input validation

Installation Steps

  1. Download the source code from GitHub
  2. Ensure Python 3.10 or higher is installed
  3. Run the main.py file in your terminal
